# Spring 2023 - SWEN 670 - Face Detector Notes

## Setup

Hopefully you already have python installed. 
That should be all you need for this to work. I have 3.11.2

You may have issues with imports...

    ```
    python.exe -m pip install --upgrade pip
    pip install imutils
    pip install opencv-python
    ```

## File Explanation

- images folder
  - Holds a bunch of different test images. They will be deleted
  and only panaromic images will be used.
- haar_face_detector.py
  - This is the file that we'll actually run, passing in an image.
  The output is the shown file with the detected faces outlined.
- haarcascade_frontalface_default.xml
  - This is the file that is generated by being trained to find 
  faces and opencv creates this file. It should not be edited.
